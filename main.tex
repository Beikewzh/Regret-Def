Continual reinforcement learning imagines an agent interacting with its world and learning from its stream of experience. 
The agent chooses actions to accumulate as much reward as possible and improve its future reward. 
To this agent, the only
A questions arises: how can we measure the performance of a continual reinforcement learning agent from its stream of experience? 


\paragraph{Learning in Any World.} Our setting considers an agent interacting with a general environment. The agent takes actions $a_t$ based on its history and the environment produces rewards and observations $(r_t, o_t)$ based on the entire history.  
No assumptions are placed on the form of the environment; it is not necessarily a MDP, POMDP, etc. Thus, we allow the agent to interact with an arbitrary stream of experience $\{o_0, r_0, a_1, o_1, r_1, a_2, o_2, r_2,...\}$ with the goal of developing an evaluation procedure for intelligent agents that can 
``continue to adapt over the course of lifelong streams of experience." \citep{silver2025welcome}.


\paragraph{Motivation: Standard measures of regret are not suitable for arbitrary streams of experience.} 
Regret is traditionally defined as $V^{*} - V^{\pi_t}$ at timestep $t$ where $V^*$ represents an ``optimal" value and $V^{\pi_t}$ represents the value of the agent's current policy.
There are two classic families of regret definitions which define $V^*$ differently: 1. \textit{Expected regret}, used in stochastic bandits and MDPs. 2. \textit{Hindsight regret}, used in adversarial bandits and online learning. 

For \textit{expected regret}, $V^*$ is the value of the optimal policy, which achieves the largest expected return in the future.
This is is an unreasonable standard for the agent to achieve in a nonstationary environment since this optimal value will depend on quantities inaccessible to the agent, such as how the environment will evolve in the future. 

For \textit{hindsight regret}, $V^*$ is defined as a counterfactual: 
What if the agent had chosen past actions according to a different policy $\pi$ instead? 
Concretely, $V^*$ is the sum of rewards that \textit{would have been accumulated} by the best policy from a given set $\Pi$ if it had been run in the past. 
For example, in classic adversarial bandits, $\Pi$ is the set of policies that always choose a single action.
Then, to achieve low hindsight regret (usually termed ``external regret"), the agent has to do (almost) as well as having chosen the best fixed action in hindsight.
In nonstationary settings or MDPs, it is more difficult to define $\Pi$ without making low regret impossible to achieve \wes{add cite}.

Some works propose other notions of regret to allow for nonstationarity in the problem settings. These include adaptive regret, tracking regret, dynamic regret among others \wes{add cites}.
At a high-level, all these regret definitions limit the amount of nonstationarity to make the problem tractable, with the exact restrictions varying from one to the other. 
For this reason, these regret definitions are unsuitable for our purpose because they explicitly or implicitly place constraints on the environment. We aim to define a performance measure suitable for \textit{any} world.

In this paper, we propose the \textit{agent-examiner framework}. An \textit{examiner} $\chi$ views the same stream of experience as the agent . 

We define a novel performance measure: The \textit{perceived regret} $\rho^\chi_t$.

\section{Formal Definitions and Assumptions}

We first define the environment and how an agent interacts with it. The environment is framed to allow maximal flexibility and impose minimal constraints on an agent's associated stream of experience. 

\wes{reorganize possibly}

\begin{definition}[Environment] The \textbf{environment} $\Env$ is defined by a tuple $(\Obs, \Actions, \Rew, \Tr, \gamma)$. The observation space $\Obs$ and action space $\Actions$ serve as the agent-environment interface. We let $\mathcal{H}$ be the space of histories, sequences $h_t =\{o_0, a_0, r_1 , o_1, ..., a_{t-2}, r_{t-1},o_{t-1} \}$ for any $t \in \mathbb{N}$. Next, $\Rew:  \Hist \times \Actions \rightarrow \Reals$ is the \textit{reward function} and $\gamma : \Hist \times \Actions$ is the \textit{discount function}, which together define the objective to maximize. Finally, $\Tr: \Hist \times \Actions \rightarrow \Obs$ defines the \textit{observation function}. \wes{maybe add something about observations}
\end{definition}


\begin{definition}[Agent and Policy]
    The \textbf{agent} $\Lambda$ chooses actions according to a \textbf{policy} $\pi$, defined as a mapping from histories to actions $\pi : \Hist \rightarrow \Actions$.  
\end{definition}
With this definition of a policy, it encompasses any algorithm that modifies the decision-making process based on past data, including memory-based policies or even parameter updates. 

The agent and environment interact in sequence, alternating between the environment producing an observation/reward and the agent choosing an action. These two steps can depend on the entire history of observations, rewards and actions up to that point in time. 


These definitions aims to capture complex streams of experience. In particular, we allow the rewards $\Rew$ and transitions $\Tr$ to depend on the entire history in an arbitrary fashion. In this way, it encompasses a large class of environments, including non-Markovian transition dynamics of any order, adaptive (or adversarial) environments and nonstationarity. 
In particular, we can model ``big worlds" \wes{add cite}, where the agent's computational capacity is tiny compared to the complexity of the world. From the perspective of the agent, the world may appear highly nonstationary and the agent cannot hope to accurately model its full complexity. 


This environment definition has certain limitations as it does not encompass continuous time environments or variable number of observations between actions. We can easily extend our proposed regret definitions to these settings by modifying the definitions of the relevant values but we focus on this setup for simplicity.
We can also augment $\Rew$, $\Tr$ and $\pi$ with an auxiliary argument $\xi \in \Omega$, which allows the rewards, observations and actions to be stochastic by treating $\xi$ as a source of randomness. 

We first define the true value and regret for an agent at a given time in a world. These quantities are unknowable to the agent but are useful to develop the theory. 


\begin{definition}[True Value and Optimal Value]
\wes{maybe call this ``environment value"}
The \textbf{true value} at time $t$ of policy $\pi$ is defined as $V_t^\pi = \Ex_{\substack{o_i \sim \Tr, r_i \sim \Rew, a_i \sim \pi \\ \text{ for } i \in \{1, 2, ...\} }}[\sum_{i=0}^\infty \gamma^i \Rew(H_{t+i}, A_{t+i}) | H_t=h]$ which is the expectation over the future according to the environment's dynamics and rewards, given the past history $h$. 
Then, the \textbf{optimal value} is defined as $V^*_t =\sup_\pi V^\pi_t$
\end{definition}

\begin{definition}[True Regret]
    The \textbf{true regret} is defined as the difference between the true optimal value and the policy value. For a policy $\pi$, the regret is defined as $\rho^*_t = V^* - V^\pi_t$
\end{definition}

\subsection{Defining Perceived Regret}


With the basic setting outlined, we can now define our proposed notion of regret. 
We consider an \textit{examiner} $\chi$ that views the same data stream as the agent at every step. 

\begin{definition}[Perceived Regret \wes{add color? highlight somehow. Make text box?}]
The \textit{examiner} $\chi$ maintains estimates of two quantities: an \emph{aspiration} $\Vhi_t$ (``the highest value it thinks is achievable'') and a \emph{lower confidence value} $\Vlow_t$ (``the value it is confident the learner achieves''). We define the \textbf{perceived regret} as $\rho^\chi_t := \Vhi_t - \Vlow_t.$ 
\end{definition}

The following assumptions make this quantity non-trivial:

\begin{assumption}[Valid lower bound]\label{a:lb}
$\Vlow_t \leq V^{\pi_t}_t$ for all $t$. The lower confidence value truly is a lower bound on the policy value. 
\end{assumption}


\begin{assumption}[Optimism]\label{a:optimistic}
$\tilde{V}^*_t \geq V^{*}_t$ for all $t$. The agent overestimates the optimal value in the environment. We can relax this condition to give different guarantees on the performance of the agent. 
\end{assumption}


\begin{proposition}[Validity]\label{prop:valid}  Under these two assumptions, $\rho^\chi_t$ upper bounds the true regret: 
\[
\rho^\chi_t = \underbrace{(\Vhi_t - V^*_t)}_{\geq\, 0} + \underbrace{(V^*_t - V^{\pi_t}_t)}_{\text{true regret}} + \underbrace{(V^{\pi_t}_t - \Vlow_t)}_{\geq\, 0} \;\;\geq\;\; V^*_t - V^{\pi_t}_t.
\]
    
\end{proposition}


Thus, driving $\rho^\chi_t \to 0$ also causes the true regret to vanish, making this definition a meaningful notion of regret.  
Additionally, these quantities are estimated exclusively from the agent's past stream of experience, without reference to external quantities from the environment.

We view these two assumptions as the minimal conditions necessary for $\rho^\chi_t$ to be useful. If either fails to hold, then $\rho^\chi_t$ may not bound the true regret and give an invalid assessment. 
However, these assumptions are not sufficient to guarantee $\rho^\chi_t$ is useful. We may have that $\rho^\chi_t$ is an upper bound but never goes to zero, even if the agent finds an optimal policy. Consider the following example.
\vspace{-2mm}
\paragraph{Example: Vacuous upper bound and examiner.} Suppose the reward is bounded in $[0, R_{max}]$. Then, suppose we have an examiner $\chi_{bad}$ that outputs $\rho^{\chi_{bad}}_t = R_{max}  / (1-\gamma)$ at every step $t$. This quantity is an upper bound on the true value (and true regret) but does not depend on the data at all. Crucially, no matter how well the agent does, the bound will never decrease and $\rho_t$ is not a useful performance measure.


We seek an examiner that refine its estimates of $\rho^\chi_t$ as it receives more information. In other words, we want an examiner that \textit{learns}. This is a tricky requirement and  we discuss how to frame this theoretically in the following section.

The examiner is faced with a learning problem of its own. 
Given the stream of observations, rewards and actions, it must estimate $\Vhi$ and $\Vlow$ and produce the regret estimate $\rho^\chi_t$. 

Broadly-speaking, we desire to have an examiner $\chi$ that produces valid (in the sense of Prop. \ref{prop:valid}) estimates while keeping the bounds as tight as possible. 
Whenever the agent is learning effectively $\rho^\chi_t$ should decrease and, if the agent has an optimal policy, the examiner should output $\rho^\chi_t=0$.
Ideally, there would be a universal examiner $\chi^*$ applicable to any agent and any world while producing effective perceived regret estimates. 

Unfortunately, this is asking for too much. The situation is similar to generalization in supervised learning, where we cannot create a universal learning algorithm that generalizes well on any data distribution.
Similarly, we prove a no-free-lunch theorem showing that it is impossible to define a ``universal" examiner capable of accurately judging all agents in all worlds and discuss the implications.

We also assume the rewards are bounded in $[0, R_{max}]$ so the true values of all policies are bounded in $[0, V_{max}]$. We define some useful quantities:
Let $V_{unif}$ be the value of the uniform policy at all future steps and define $\rho_{bound} := V_{max} - V_{unif}$. This quantity is an upper bound on useful perceived regret estimates since, if the true regret is larger than $\rho_{bound}$, the agent is worse than the uniform random policy.

Similar to no-free-lunch theorems in supervised learning, which tell us that over all data distributions, no model can outperform random guessing on the population loss, we find that it is impossible to produce valid non-vacuous bounds on all environments. 

where we cannot expect a learning algorithm to generalize well for all data distributions, we cannot expect the examiner to accurately assess any learner. 
The examiner must make some guess about the structure of the environment to account for possible futures and their associated rewards.  
This guess depends on the various modeling biases baked into the examiner. 
For example, the examiner may assume that the world is Markovian and produce its estimates in accordance with this assumption. This may be correct in certain environments and incorrect in others. 
The examiner's inductive bias also matters as the agent may enter previously unseen parts of the world and thus the examiner's generalization properties matter too.

Overall, we see that the examiner faces a learning problem at least as difficult as supervised learning, combining standard predictions on examples with a temporal and decision-making component. 
As such, it is expected that similar no-free-lunch theorems apply to the examiner. 
This means there are no ``universally" best examiners---we must make do with the examiners we believe are best fit for the environment at hand and cannot give general guarantees on their evaluation performance. 
Next, we discuss some choices of modeling biases we may endow the examiner with and reasonable criteria we could expect.

Since universal examiners are impossible, we need to make tradeoffs and choose properties we may want an examiner to have. To help us decide, we may prefer examiners that are known to work in specific types of environments of interest. 
We can assess an examiner by how it behaves in certain families of environments when run on known, optimal learning agents.

For example, in stochastic bandits and finite MDPs, there are known provably-optimal learning algorithms, achieving the best (true) expected regret possible. These agents will shrink the regret per step towards zero. 
A ``good" examiner, when run on these learning agents, should also reduce its perceived regret towards zero as the agent converges towards an optimal policy. 

We may be able to create more flexible examiners, capable of dealing with larger families of environments, by considering environments where provably-optimal learning agents exist. Drawing from the vast reinforcement learning theory literature, theorists have crafted increasingly general conditions where provably low regret is achievable. For example, tabular MDPs are a special case of linear MDPs \citep{jin2020provably}, which are in turn subsumed by low Bellman rank MDPs \citep{jiang2017contextual} or low Eluder dimension \citep{russo2013eluder}. These can be  further generalized with concepts such as the Bellman-Eluder dimension \citep{jin2021bellman}, bilinear classes \citep{du2021bilinear} and many alternative concepts \citep{ayoub2020model, sun2019model}. One of the most general family of environments where low regret is possible are those with low Decision-Estimation Coefficient \citep{foster2021statistical}. 
Whenever we have a provably optimal agent for some environment family, we can use this as a benchmark to test candidate examiners. In these cases, we can require that the examiner produces a perceived regret decreasing to zero at the similar rate as the true regret. 

It is important note that, in light of the no-free-lunch theorem, it will be impossible to expand the family of environments ad infinitum. The examiner will always be unreliable in some world. 
Certain examiners may also provide invalid perceived regret, which do not bound the true regret. In a general environment, we may not have any formal guarantee, similar to generalization in supervised learning. In practice, on a unknown environment, we  forgo any formal guarantees. 
% These are all modeling decisions that a use must make. 

Without a prior knowledge of the environment, we can mitigate the possibility of poor examiner by utilizing multiple examiners in parallel on a single agent's stream of experience, resulting in several performance estimates. These can aggregated or used a distribution of assessments. Similar to ensembles, we may want to choose a diverse set of examiners, with different modeling biases and learning mechanisms. 
We can also consider pretrained models (e.g. foundation models) that have assimilated a large corpus of data. This can lead to complex inductive biases within the examiner that could potentially help it produce more useful perceived regret values. 

In the next section, we illustrate some simple experiments where the appropriate choice of examiner for an agent and environment leads to meaningful perceived regret throughout the agent's learning process.

\section{Perceived regret in practice}
\label{sec:practice}

Having argued that no universal examiner exists, we now ask how an appropriately-chosen examiner behaves on a learning agent in a known world. We use two multi-armed bandit testbeds where the true regret can be computed in closed form, so that the perceived regret $\rho^\chi_t$ can be read against the ground truth. The stationary testbed is a $10$-arm bounded-Gaussian bandit with one rewarding arm ($\mu{=}0.75$), nine sub-optimal arms ($\mu{=}0.25$), noise $\sigma{=}0.25$ and horizon $T{=}10\mathrm{k}$. The switching testbed reuses the same arm distribution but rotates the rewarding arm every $10\mathrm{k}$ steps for five segments, giving $T{=}50\mathrm{k}$. For each world, we pair a \textit{good} agent whose inductive bias matches the world with a \textit{bad} agent whose bias does not, and run each agent under a \textit{good} and a \textit{bad} examiner constructed in the same way. The good and bad roles are exactly mirrored across the two worlds: in the stationary world, ``good'' means full-memory and ``bad'' means a sliding window; in the switching world, the assignment is reversed. All curves are averaged over $100$ seeds.

In the stationary world (Figure~\ref{fig:practice_main}a), the good agent is UCB1 and the bad agent is $\varepsilon$-greedy with $\varepsilon{=}0.1$. The good examiner maintains Hoeffding upper and lower confidence values with full memory, while the bad examiner uses a sliding-window variant and so behaves as if the world were non-stationary. UCB1 drives the true regret toward zero, and the matched examiner's $\rho^\chi_t$ shrinks to it from above, recovering Proposition~\ref{prop:valid} empirically. On the $\varepsilon$-greedy agent, the true regret plateaus at the sub-optimality gap induced by constant random exploration, and the matched examiner reports a $\rho^\chi_t$ that plateaus at the same level. The sliding-window examiner, in contrast, never accumulates enough samples to tighten its bracket and so reports a $\rho^\chi_t$ that stays near its initial vacuous value even after the good agent has effectively solved the problem: a valid but uninformative upper bound, of the kind described in the example of Section~\ref{sec:intro}.

The switching world (Figure~\ref{fig:practice_main}b) makes the symmetric point. Now the good agent is an oracle reset-UCB that knows the switch times, the bad agent is plain UCB1, and the bad examiner is the full-memory Hoeffding bracket that worked well in the stationary world. At each switch, the true regret jumps and then decays as the matched agent re-explores; the matched examiner's $\rho^\chi_t$ tracks this trajectory, spiking and recovering segment by segment. The bad examiner, by holding on to data from previous segments, keeps $\Vhi$ artificially low and $\Vlow$ artificially high relative to the current segment and so reports a $\rho^\chi_t$ that drifts toward zero between switches, even when the underlying agent is incurring substantial true regret. The mis-specified examiner therefore violates the optimism assumption (\ref{a:optimistic}) and produces an invalid assessment. The same examiner applied to UCB1 mis-reports its persistent regret in the same direction. Together, the two panels exhibit both failure modes anticipated by the theory: a vacuous-but-valid bracket in (a) and a tight-but-invalid bracket in (b).

These observations suggest a simple way to use perceived regret on a stream whose dynamics are not known in advance. Rather than committing to a single examiner, one can instantiate a small pool of examiners with diverse inductive biases, run them in parallel on the same stream and report their $\rho^\chi_t$ together with the agent's behaviour. Agreement across the pool is evidence that the regret estimate is meaningful, while disagreement is itself a diagnostic that the choice of examiner, rather than the agent, requires revisiting. We use this pattern in the rest of this section as we vary the relative capacity of the agent and examiner.

\textbf{Limitations.} The perceived regret is produced by an examiner, which is also a learning agent (that cannot influence the future). When deployed alongside the agent, there are no guarantees that its perceived regret is actually meaningful and it could be wildly incorrect. 
This may be an unavoidable price of seeking to evaluate agents in arbitrary worlds---the general applicability impacts the reliability of the assessment. This issue is akin to the problem of generalization. When an agent encounters a truly novel situation, there cannot be any guarantees and we can only hope it generalizes correctly. 



\paragraph{Models evaluating other models.} The definition of perceived regret is similar in spirit to previously-used performance measures in the machine learning community. Generative models are one area where it has been common to assess the performance of one model with others. 
One example is the Inception score used as a performance metric to measure diversity and fidelity of a model's generated images by passing them through the Inception network. Another is CLIPScore in a similar context for test-to-image generation, utilizing the CLIP model as an evaluator. 
More recently, LLM-as-judge is a common framework used to automatically assess the outputs of a given LLM by feeding it to  multiple other LLMs.
LLMs have also been used to inspect their own outputs, critiquing and refining them to produce higher-quality answers \wes{add cite}.
