\section{Introduction}
\label{sec:intro}

Reinforcement learning carries an unexamined assumption about evaluation: that an agent is graded from outside, against an optimum $V^{*}$ that a model, a prior, or a fixed comparator supplies \citep{lai1985asymptotically,jaksch2010ucrl2}. This is harmless when the world is small enough to model. Continual reinforcement learning (CRL) rests on the opposite premise \citep{ring1994continual,abel2023definition}. Under the big-world hypothesis the agent is far smaller than its world, cannot represent or retain a fixed optimal solution, and never stops learning \citep{javed2025bigworld,lewandowski2025bigger,dohare2024plasticity}. The external yardstick then fails twice. At run time the agent has no source for $V^{*}$. And even a known $V^{*}$ is the wrong target, since a bounded agent should be judged against the best use of its own resources rather than against an optimum it could never reach \citep{simon1955behavioral,russell1995bounded,lieder2020resource,abel2024dogmas}.

Our position is that if a continual agent cannot be graded from outside, its only honest measure of progress is one it computes for itself, from its own stream. We make that measure an explicit object. Alongside the acting learner, an internal examiner forecasts an aspiration $\Vhi$ for the value it believes attainable and a guarantee $\Vlow$ for the value it is sure of; their gap, the examiner regret $\rho_t = \Vhi - \Vlow$, is the agent's self-assessed room to improve. We argue that this self-evaluation certifies nothing on its own, that a soundness condition turns it into a certificate of the regret the agent can actually reduce, and that in a big world this certificate is never free, because what the agent cannot see is not identifiable from its stream at any budget. The load-bearing consequence is a design principle: to be trustworthy, the examiner must out-learn the policy it grades, since judging whether one could have done better means reasoning about choices never tried, which a single acting policy cannot supply. We claim no new algorithm or rate. A bandit runs as the worked example throughout, and the same bandit recovers the classical confidence-bound machinery as a special case.

\section{Where the Standard Measure Fails}
\label{sec:fails}

The case for a new measure is concrete: in the settings CRL is meant to capture, the external comparator is not merely unknown but ill-posed. Three examples make this precise.

\textbf{A growing set of behaviours.} On an open-ended stream new options keep appearing. Standard regret asks for the value of the best policy, but when the set of behaviours is itself growing, that supremum is undefined at any finite time, and the regret returns no number.

\textbf{An option never sampled.} Even with a fixed action set, the value $V^{*}$ of the best behaviour depends on outcomes the agent may never observe. If a better option is never tried, $V^{*}$ is a function of parts of the world that leave no trace in the stream, so it is not identifiable, and the regret is uncomputable by the agent regardless of its compute.

\textbf{A moving target.} When the best action drifts over time, a single fixed comparator is meaningless. Dynamic and non-stationary regret restore a moving comparator \citep{wei2021master,morrill2021hindsight}, but they still presume one can name the best policy in hindsight, which is exactly what a big world denies.

In all three the failure is the same: the comparator is external and unavailable. The agent can still answer a question posed entirely within its own stream, namely how much better it believes it could be doing than it is. The rest of the paper makes that question precise and asks what its answer is worth.

\section{Examiner Regret}
\label{sec:defs}

We work over a single, never-resetting stream of experience, with no Markov, episodic, or stationarity structure assumed. A history is $h_t = (o_0,r_0,a_1,\dots,a_t,o_t,r_t)$ under an unknown law $\mu$, and a policy $\pi$ maps histories to actions with $H$-step value $f^{\pi}_t = \Ex_{\mu,\pi}[\sum_{s=t+1}^{t+H} r_s \mid h_t]$. We assume bounded rewards $r_s \in [0,\Rmax]$ \textbf{(A1)}; a fixed look-ahead $H$ \textbf{(A2)}; a capacity-bounded learner representing policies within $K_L$ bits, giving the representable set $\PiK$ with $|\PiK| \le 2^{K_L}$ \textbf{(A3)}; an examiner carrying an inductive bias, a model class $\ModE$ such as a concentration rule or a prior \textbf{(A4)}; high-probability guarantees over the stream \textbf{(A5)}; and that every output of learner and examiner is measurable with respect to the observed history $\sigma(h_t)$ \textbf{(A6)}, that is, the agent knows only its own stream, whatever its compute.

Because the agent cannot represent a global optimum, we judge it against the best behaviour it can itself express. The within-capacity optimum is $f^{K_L}_t = \sup_{\pi \in \PiK} f^{\pi}_t$, and the structural gap $\Dstruct_t = f^{*}_t - f^{K_L}_t \ge 0$ is the irreducible price of being small, kept visible rather than hidden. The agent is a pair $(\mathcal{L},\mathcal{E})$. The learner $\mathcal{L}$ emits the policy $\pi_t \in \PiK$ it acts with and learns only about that policy. The examiner $\mathcal{E}$ is a predictive model that, from the history, outputs an aspiration $\Vhi$ and a guarantee $\Vlow$ with $\Vhi \ge \Vlow$. Setting $\Vhi$ means forecasting the value of policies the learner never ran, so the examiner must learn about a family of behaviours, not the one acted, in the spirit of general value functions \citep{sutton2011horde,modayil2014nexting} and off-policy prediction \citep{precup2001offpolicy,suttonprecupsingh1999}, with $\ModE$ the bias that lets it generalize beyond what it has tried \citep{osband2013psrl}.

\begin{definition}[Examiner regret]
\label{def:exreg}
$\rho_t = \Vhi - \Vlow \ge 0$, with $\Rex_T = \sum_{t=1}^T \rho_t$. It is computed from the history and the examiner alone; it never refers to $\mu$ or to any optimum.
\end{definition}

\section{From a Cheap Signal to a Certificate, and What It Recovers}
\label{sec:certificate}

The definition is cheap by design, and the first thing to say is what that cheapness costs.

\begin{proposition}[A self-report carries no guarantee]
\label{prop:nothing}
There is an examiner reporting $\rho_t \equiv 0$ for every learner and world, namely the one that outputs $\Vhi = \Vlow = 0$. Two agents with this identical self-report can differ arbitrarily in true performance, one optimal within its capacity and the other worse by a constant at every step.
\end{proposition}

So trust cannot live in $\rho_t$. It must come from a condition tying the examiner's two numbers to real value. Call the examiner sound with slack $(\delta_t)$ if, with high probability, the guarantee is a true floor, $\Vlow \le f^{\pi_t}_t$, and the aspiration is a true ceiling for the within-capacity optimum, $\Vhi \ge f^{K_L}_t - \delta_t$. Soundness immediately rules out the empty report of Proposition~\ref{prop:nothing}. It also turns $\rho_t$ into a bound on the regret a bounded agent can drive down. Telescoping $f^{K_L}_t - f^{\pi_t}_t = (f^{K_L}_t - \Vhi) + (\Vhi - \Vlow) + (\Vlow - f^{\pi_t}_t)$ and summing,
\[
\sum_{t=1}^T \big(f^{K_L}_t - f^{\pi_t}_t\big) \;\le\; \Rex_T + \sum_{t=1}^T \delta_t,
\]
with the global optimum adding only the irreducible $\sum_t \Dstruct_t$. This is the optimism and performance-difference identity relabeled \citep{kakade2002cpi,jaksch2010ucrl2}, not a new rate; its value is interpretive. Meeting the ceiling, though, demands more learning than acting: the learner extracts one policy's value from the stream, whereas a sound examiner must extract the value of a whole family, including behaviours never run. That is the precise sense in which the examiner must out-learn the learner.

\paragraph{What it recovers.} The construction is not exotic; in the coverable case it is the classical confidence-bound theory under a new name. Take a bandit, our stream with $|\mathcal{S}|=1$ and $H=1$, with $N$ arms of mean $\mu_i$, all $\tfrac12$ except one special arm at $\mu^{\star} = \tfrac12 + \epsilon$. Here $f^{\pi}_t = \mu_i$ and $f^{K_L}_t = \mu^{\star}$. Writing $\hat\mu_i$ for the empirical mean of arm $i$ pulled $n_i$ times and $c_i = \sqrt{2 \ln t / n_i}$ for the Hoeffding radius (the standard choice making the bound hold over all arms and times \citep{auer2002ucb}), the examiner is exactly UCB: $\Vhi = \max_i(\hat\mu_i + c_i)$ and $\Vlow = \hat\mu_{a_t} - c_{a_t}$, so $\rho_t$ is the confidence width. Two facts follow. The agent's regret against its best arm, $\sum_t(\mu^{\star} - \mu_{a_t})$, is the classical pseudo-regret \citep{lai1985asymptotically}, but computing it needs the hidden $\mu^{\star}$, so only an outside analyst can, whereas $\rho_t$ needs no hidden mean. And summing the widths gives $\Rex_T = O(\sqrt{NT\log T})$, the standard UCB rate. The point is the combination: when the agent can cover its arms the slack vanishes, $\rho_t$ shrinks, and the agent certifies how far it is from its best arm without ever knowing which arm that is.

\section{The Price: No Free Self-Evaluation in a Big World}
\label{sec:nofree}

Coverage is exactly what a big world denies, and the reason is informational, not computational. Suppose $N > T$: the learner pulls at most $T$ arms, so some arm is never tried, and the special arm can hide there. Two worlds that differ only on an unpulled arm induce the same distribution over observed histories, so by \textbf{(A6)} every $\sigma(h_t)$-measurable procedure, of any complexity, behaves identically in both, while their $f^{K_L}$ differ by $\epsilon$. This is Le Cam's two-point bound with total variation zero on the observed history \citep{tsybakov2009}: no procedure can resolve the difference. Soundness then forces $\Vlow \le \tfrac12$ and $\Vhi \ge (\tfrac12 + \epsilon) - \delta_t$, hence
\[
\rho_t \;\ge\; \epsilon - \delta_t, \qquad\text{so}\qquad \Rex_T \;\ge\; \epsilon T - \sum_{t \le T} \delta_t,
\]
unless the agent closes one of two gaps: a coverage gap (try every arm, $T \ge N$) or a bias gap ($\ModE$ already contains the distinction, which it cannot check against arms it has not resolved). The same impossibility is the explore-or-assume dichotomy and the realizability requirement priced for decades \citep{kearns2002e3,brafman2002rmax,azar2017minimax,lai1985asymptotically}. We invoke it not as a new hardness result but to price the position: a near-zero $\rho_t$ in a genuinely big world is unearned confidence, not completed learning. Extra budget tightens estimates only within what the data and bias already determine, and cannot manufacture information about an arm never pulled, so a computationally lavish examiner is still bound by it.

Figure~\ref{fig:bandit} shows both faces. In the big-world bandit (panel A) a sound examiner keeps $\rho_t$ large and honest while a biased examiner that assumes a shared mean collapses $\rho_t$ below the true regret, a false report that the agent is done. Under non-stationarity (panel B), where the special arm switches periodically, the within-capacity optimum keeps moving and consistency cannot hold cheaply: the sound self-report stays a valid certificate above the true regret, while the biased one again fails to bound it. This is the regime CRL lives in, and it is where the price is steepest.

\section{Building an Examiner that Out-Learns the Actor}
\label{sec:practical}

The principle is constructive, because both of the examiner's numbers are read off methods already in use: $\Vhi$ is any optimistic value estimate, such as a UCB index \citep{auer2002ucb}, optimistic value iteration \citep{jaksch2010ucrl2}, an ensemble upper quantile \citep{osband2016bootstrapped}, or a posterior upper quantile \citep{osband2013psrl}, and $\Vlow$ any pessimistic one, such as a conservative critic \citep{kumar2020cql,jin2021pessimism}. The harder question is how the examiner can be better informed than the learner on the same stream, and there are three answers, none of which is simply a better prior. It learns off-policy about a family of policies the learner never runs, through off-policy temporal-difference learning and general value functions \citep{sutton2011horde,precup2001offpolicy,modayil2014nexting}, which is exactly what setting $\Vhi$ demands. It may carry more capacity and a faster timescale than the real-time learner \citep{pinto2018asymmetric,konda2000actorcritic,borkar1997twotimescale}. And it is trained to a calibration target \citep{foster1998calibration}, so its slack $\delta_t$ shrinks on the covered region and the question ``is the examiner learning?'' becomes operational. None of the three manufactures information, so the lead is real but capped by coverage, the lower bound turned into a design rule: separate the critic from the actor and give it its own larger budget, off-policy targets, and a calibration loss. Splitting a total budget $K = K_L + K_E$ between reaching (coverage) and judging (a richer model) is the open problem the position opens. For a deployed agent one cannot grade against an optimum, such as a recommender or a robot, $\rho_t$ is then an online read of self-assessed room to improve, and the price is the operator's warning: a confidently small $\rho_t$ may be unearned, so apparent convergence should trigger exploration or external testing rather than trust.

\section{Limitations}
\label{sec:limits}

The content lives in soundness, not in $\rho_t$, and soundness is extrinsic, requiring the examiner to carry a model class and hence a prior. The bridge inequality is a decomposition, the optimism gap relabeled, not a new rate. Our tight, positive result is the coverable bandit; soundness of a biased, capacity-bounded examiner under genuine non-stationarity, and the under-capacity case $\Dstruct_t > 0$, are not established here \textbf{[GAP]}. The figure is an illustration on a synthetic bandit, not evidence of a general guarantee. The impossibility is worst-case and not itself novel, and we have not characterized which $\ModE$ avoid the bias gap on which world families. The out-learning principle is a design argument grounded in existing off-policy and asymmetric-critic methods, not a theorem that an off-policy examiner attains soundness faster than the learner attains optimality \textbf{[GAP]}. We propose one coupling, feeding $\rho_t$ back as an intrinsic reward, but do not analyze the joint dynamics.

\section{Related Work}
\label{sec:related}

\textbf{Foundations.} \citet{ring1994continual,abel2023definition,abel2024dogmas,abel2020bounded} develop CRL and the agent-centric view; bounded optimality and resource-rational analysis \citep{simon1955behavioral,russell1995bounded,lewis2014computational,lieder2020resource} motivate our comparator $f^{K_L}$ and the reward hypothesis \citep{silver2021reward} the goal it tracks, while the big-world hypothesis \citep{javed2025bigworld,lewandowski2025bigger,dohare2024plasticity} drives the removal of the external yardstick. \textbf{Machinery.} General value functions and off-policy prediction \citep{sutton2011horde,modayil2014nexting,suttonprecupsingh1999,precup2001offpolicy} and asymmetric and two-timescale actor critic \citep{pinto2018asymmetric,konda2000actorcritic,borkar1997twotimescale} let the examiner learn about behaviours it does not enact; optimistic and pessimistic value learning \citep{kearns2002e3,brafman2002rmax,jaksch2010ucrl2,auer2002ucb,strehl2009pacmdp,azar2017minimax,kakade2002cpi,kumar2020cql,jin2021pessimism,osband2016bootstrapped} supply its two numbers, and the two-point method \citep{lai1985asymptotically,tsybakov2009} its price. \textbf{Closest.} Bayesian and capacity-limited RL \citep{osband2013psrl,russo2017ids,lu2021bitbybit,arumugam2024capacity} and model-value inconsistency \citep{filos2022modelvalue} read a value gap as uncertainty; self-assessment and calibration \citep{schmidhuber2010formal,oudeyer2007intrinsic,bellemare2016count,pathak2017curiosity,burda2019rnd,foster1998calibration,li2011kwik,morrill2021hindsight} bear on honest progress. Across all of these the evaluator is external or fused with the policy, whereas we make it an explicit, continually-learning, capacity-bounded module and ask what its self-report is worth.

\section{Conclusion}
\label{sec:conclusion}

Continual learning has no examiner outside the agent, so the regret that matters is the one a bounded internal examiner can certify. That self-evaluation is the only honest internal measure of progress, is meaningful only under soundness, and meeting soundness forces the examiner to out-learn the policy it grades. In a big world this is never free: what the agent cannot see is not identifiable from its stream at any budget. The position turns evaluation in CRL into a budgeting problem between acting and judging, makes ``is the agent still learning?'' the operational question answered by a sound, shrinking $\rho_t$, and warns against trusting an agent that reports it has finished.
