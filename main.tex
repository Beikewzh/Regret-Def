\begin{abstract}
Continual reinforcement learning considers an agent receiving and learning from a stream of experience. The agent only learns about its world through these experiences and aims to maximize its accumulated reward. A fundamental problem is to evaluate this agent using only its stream of experience, without assuming a particular structure of the world.
We outline a novel approach to this problem: Define an \textit{examiner} that observes the same stream of experience and, at every timestep, computes a \textit{perceived regret}, representing the  agent's suboptimality gap in value from the examiner's perspective.
We empirically demonstrate how perceived regret can be a useful performance measure, applicable to arbitrary streams of experience.
Theoretically, we prove that there is no universal examiner, capable of accurately assessing all agents in any world but specific modeling biases can enable success in particular environments.
Further experiments show the flexibility of this approach and explore its nuances. 

\end{abstract}
