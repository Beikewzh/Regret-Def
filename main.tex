\section{Introduction}
\label{sec:intro}

\section{Position}

Standard reinforcement learning grades an agent from the outside, against the best possible score $V^{*}$. That works only when something outside the agent can know $V^{*}$. In a big world it cannot: the agent is far smaller than its world, no single best policy can even be written down, and learning never stops, so $V^{*}$ has no source and no fixed value.

Our position is that the only honest way for such an agent to tell whether it is still improving is to grade itself, from its own experience. We call that self-grade the \emph{examiner regret} $\rho_t$. For the self-grade to be trustworthy, the part of the agent that does the grading must learn harder than the part that acts, because asking ``could I have done better?'' means judging choices the agent never tried. This is never free: an honest self-grade costs either trying everything or trusting an assumption the agent cannot check. We make that cost explicit rather than hide it.

\section{Definition}
\label{sec:defs}

We take the agent-centric, big-world view of continual reinforcement learning. The big-world hypothesis says the agent is far smaller than its environment, so it cannot represent or store a single best policy \citep{javed2025bigworld,lewandowski2025bigger}. Learning therefore never stops, and the goal is to keep adapting rather than to find one fixed solution \citep{ring1994continual,abel2023definition,abel2024dogmas}. A small agent should be judged against the best it can do with its own limited resources, not against an external optimum \citep{simon1955behavioral,russell1995bounded,lieder2020resource}. This is why our comparator is the within-capacity optimum $f^{K_L}$ rather than $V^{*}$: examiner regret measures progress toward the reward the agent can actually reach \citep{silver2021reward}.

We work over a single, never-resetting stream of experience, with no Markov, episodic, or stationarity structure assumed. We want a measure of learning the agent can compute from its own state, a condition under which that measure is trustworthy, and the limit beyond which it cannot be.

\paragraph{Assumptions.} \textbf{(A1)} rewards are bounded, $r_s \in [0,\Rmax]$. \textbf{(A2)} value is measured over a fixed $H$-step look-ahead. \textbf{(A3)} the learner is capacity-bounded, representing policies within $K_L$ bits, which fixes the representable set $\PiK$. \textbf{(A4)} the examiner is a predictive model carrying an inductive bias, a model class $\ModE$ (a concentration rule or a prior), that lets it forecast the value of behaviours not yet tried. \textbf{(A5)} guarantees are high-probability statements over the randomness of the stream.

The four definitions below build up in order: the value of a behaviour (Def.~\ref{def:basic}), the best the agent could itself achieve (Def.~\ref{def:capcomp}), the split of the agent into an acting \emph{learner} and a grading \emph{examiner} (Def.~\ref{def:agent}), and the self-grade those two produce (Def.~\ref{def:exreg}).

\begin{definition}[Stream, policy, value]
\label{def:basic}
A \emph{history} is $h_t = (o_0,r_0,a_1,\dots,a_t,o_t,r_t)$, with observations $o_s$, rewards $r_s \in [0,\Rmax]$, actions $a_s$; the underlying law is $\mu$. A \emph{policy} $\pi$ maps histories to action distributions, with value $f^{\pi}_t = \Ex_{\mu,\pi}\!\big[\sum_{s=t+1}^{t+H} r_s \mid h_t\big]$.
\end{definition}

\begin{definition}[What the learner could have done]
\label{def:capcomp}
The learner represents policies within $K_L$ bits; write $\PiK$ for this representable set. The \emph{within-capacity optimum} is $f^{K_L}_t = \sup_{\pi \in \PiK} f^{\pi}_t$, and the \emph{structural gap} to the global optimum is $\Dstruct_t = f^{*}_t - f^{K_L}_t \ge 0$. We compare the agent to $f^{K_L}_t$, the best it could itself express, and keep $\Dstruct_t$ visible as the irreducible price of being small.
\end{definition}

\begin{definition}[Learner and examiner]
\label{def:agent}
An agent is a pair $(\mathcal{L},\mathcal{E})$. The \emph{learner} $\mathcal{L}$ emits the policy $\pi_t \in \PiK$ it acts with, and learns only about that policy. The \emph{examiner} $\mathcal{E}$ is a \emph{predictive model}: from the history it outputs an \emph{aspiration} $\Vhi$ and a \emph{guarantee} $\Vlow$, with $\Vhi \ge \Vlow$. Setting $\Vhi$ requires forecasting the value of policies the learner did not run, so the examiner must learn about a whole family of behaviours, not the one acted. In the spirit of general value functions \citep{sutton2011horde,modayil2014nexting}, temporally abstract predictions \citep{suttonprecupsingh1999}, and off-policy prediction \citep{precup2001offpolicy}, $\Vhi$ and $\Vlow$ are value predictions the agent learns from its own stream, and the examiner's \emph{inductive bias}, the model class $\ModE$ of environments it considers possible, the concentration rule, or the prior of its predictors, is what lets it generalize to behaviours it has not tried. A Bayesian posterior over a model class is one instance, not the only one \citep{osband2013psrl}.
\end{definition}

\begin{definition}[Examiner regret]
\label{def:exreg}
$\rho_t = \Vhi - \Vlow \ge 0$, with $\Rex_T = \sum_{t=1}^T \rho_t$. It depends only on the history and the examiner; it never refers to $\mu$ or to any optimum.
\end{definition}

\section{A Worked Case: The Examiner in a Bandit}
\label{sec:bandit}

The simplest world in which the definitions become concrete has no state: a multi-armed bandit is the stream of Section~\ref{sec:defs} with $|\mathcal{S}|=1$ and $H=1$. There are $N$ arms; pulling arm $i$ returns a reward $r\in\{0,1\}$ with unknown mean $\mu_i$, so $\Rmax=1$. Every arm pays $\tfrac12$ except one \emph{special} arm with mean $\mu^\star=\tfrac12+\epsilon$. With $H=1$ the value of ``pull arm $i$'' is just $\mu_i$, the best representable behaviour is ``pull the best arm,'' so the within-capacity optimum is $f^{K_L}_t=\mu^\star$ and the structural gap is $\Dstruct_t=0$.

\paragraph{The examiner's two numbers.} Let $n_i$ be how many times arm $i$ has been pulled and $\hat\mu_i$ its average reward so far. By Hoeffding's inequality the average of $n_i$ rewards in $[0,1]$ deviates from the true mean $\mu_i$ by more than $c$ with probability at most $2e^{-2 n_i c^2}$; setting $c_i=\sqrt{2\ln t / n_i}$ makes this probability $2t^{-4}$, small enough to hold simultaneously over all arms and all times $t$, which is why this is the standard radius \citep{auer2002ucb}. The examiner takes an optimistic ceiling over all arms and a pessimistic floor on the played arm $a_t$,
\[
\Vhi=\max_i\big(\hat\mu_i+c_i\big), \qquad \Vlow=\hat\mu_{a_t}-c_{a_t},
\]
so the examiner regret $\rho_t=\Vhi-\Vlow$ (Definition~\ref{def:exreg}) is the confidence width. It is computed only from the observed counts $n_i$ and averages $\hat\mu_i$; it never uses the hidden means $\mu_i$ or $\mu^\star$.

\paragraph{A sanity check: it matches what the field already trusts.} A new definition should reduce to known quantities in a known case, and here it does, in three steps. \emph{First}, the agent's regret against its own best arm is $\sum_t(f^{K_L}_t-f^{\pi_t}_t)=\sum_t(\mu^\star-\mu_{a_t})$, the classical bandit pseudo-regret \citep{lai1985asymptotically}; computing it needs the hidden $\mu^\star$, so only an outside analyst can. \emph{Second}, the agent's own self-grade $\sum_t\rho_t$ needs no hidden mean, and it grows at the rate $O(\sqrt{NT\log T})$: each pull of an arm adds a width $\sqrt{2\ln t/n_i}$, the widths of an arm pulled $n_i$ times sum to about $2\sqrt{2 n_i\ln T}$, and summing over arms with $\sum_i n_i=T$ and Cauchy--Schwarz gives the bound, which is exactly the standard UCB rate \citep{auer2002ucb}. \emph{Third}, and this is the point, when the examiner is sound the self-computable $\sum_t\rho_t$ sits above the true pseudo-regret: the agent certifies how far it is from its best arm \emph{without ever knowing which arm that is}. The examiner is nothing exotic here; its two numbers are UCB's own upper and lower confidence bounds, renamed.

\paragraph{Why it stays honest, and the price.} An arm never pulled has $n_i=0$, so its radius $c_i$ stays at its maximum and its optimistic index stays high: the ceiling $\Vhi$ cannot fall, and the examiner cannot claim ``no room to improve'' about an arm it has not inspected. That keeps a small $\rho_t$ trustworthy when the agent can cover its arms. But when there are more arms than steps ($N>T$), some arm is never pulled, and if the special arm hides there the examiner must keep $\rho_t\ge\epsilon$: it cannot honestly certify it is done. An honest, vanishing self-report is therefore not free in a big world.

\paragraph{How to use it.} Without any external optimum, $\rho_t$ serves as an online \emph{monitor} of self-assessed room to improve; comparing the $\rho_t$ of two algorithms on one stream \emph{ranks} which is nearer its own reachable optimum; and a low $\rho_t$ can be \emph{audited} on the covered arms, with the big-world case marking when no audit is possible and apparent convergence should trigger more exploration rather than trust.
