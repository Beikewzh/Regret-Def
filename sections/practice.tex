\section{Perceived regret in practice}
\label{sec:practice}

We illustrate how perceived regret behaves when the examiner's inductive bias is, or is not, matched to the world. We use bounded-Gaussian bandit testbeds in which the optimal policy and the true regret are both known, so $\rho^\chi_t$ can be read against ground truth; full settings are deferred to Appendix~\ref{app:details}. Throughout, our expectation follows the theory of Section~\ref{sec:nfl}: an examiner reports a valid and tight $\rho^\chi_t$ when its assumptions match the structure that drives regret, and it fails in a predictable direction when they do not.

\begin{figure}[t]
\centering
\includegraphics[width=\textwidth]{image/fig12_combined.pdf}
\caption{Perceived regret $\rho^\chi_t$ versus true regret on the agent/examiner grid, in (a) a stationary and (b) a switching bounded-Gaussian bandit (dotted: segment boundaries). Each world pairs a matched and a mismatched examiner with a well-adapted and a poorly-adapted agent. Curves are the mean over $100$ seeds; shaded bands are $95\%$ bootstrap intervals.}
\label{fig:practice_main}
\end{figure}

\textbf{Matched examiners track true regret.} In the stationary world (Figure~\ref{fig:practice_main}a), the matched examiner's $\rho^\chi_t$ decays toward zero on a converging agent and settles at the residual sub-optimality gap on an agent that keeps exploring, mirroring the true regret in both cases. In the switching world (Figure~\ref{fig:practice_main}b), a reset-aware examiner spikes at each change and recovers between changes, following the true regret of an agent that re-adapts, and it correctly reports persistent regret for an agent that assumes the world is static.

\textbf{Mismatched examiners fail predictably.} An examiner that assumes the wrong dynamics fails in the direction its bias dictates. A stationarity-assuming examiner in the switching world lets $\rho^\chi_t$ drift toward zero between changes even while the agent incurs large true regret, an optimistic and invalid assessment. Conversely, an examiner that always expects change inflates $\rho^\chi_t$ on an agent that has already converged. These are the two practical faces of the no-free-lunch result: validity (Assumptions~\ref{a:lb}, \ref{a:optimistic}) and tightness can each be lost when the examiner is incompatible with the stream.

\subsection{Relative capacity between learner and examiner}
\label{sec:capacity}

A second axis is representational capacity: how finely the examiner, or the agent, resolves the action space. We probe it by letting one party distinguish only a coarse grouping of the arms while the other stays fine-grained. We expect $\rho^\chi_t$ to stay valid and tight only when the examiner is at least as fine-grained as the distinctions that drive regret, and, when the agent is instead the coarser party, to remain valid while exposing the agent's structural limitation. Figure~\ref{fig:capacity} confirms both in the switching world; the milder stationary case is in Appendix~\ref{app:capacity_stationary}.

\begin{figure}[t]
\centering
\includegraphics[width=\textwidth]{image/fig4_capacity_forward.pdf}\\[6pt]
\includegraphics[width=\textwidth]{image/fig4_capacity_reverse.pdf}
\caption{Relative capacity in the switching bandit, one panel per resolution $G$ (the number of arm groups the coarse party can distinguish). \textbf{Top (forward).} A fine-grained agent assessed by an examiner that resolves only $G$ groups: a coarse examiner is blind to within-group changes and under-reports, with $\rho^\chi_t$ falling below the true regret (red); only the fully resolved examiner tracks the truth. \textbf{Bottom (reverse).} An agent restricted to $G$ groups assessed by a fully resolved examiner: $\rho^\chi_t$ stays valid and plateaus just above the structural gap (labelled) that the restricted agent cannot close, while the fully resolved agent drives both regrets toward zero. Red: true regret; blue: $\rho^\chi_t$.}
\label{fig:capacity}
\end{figure}

\textbf{An under-resolved examiner under-reports.} When the agent is fine-grained but the examiner aggregates arms (Figure~\ref{fig:capacity}, top), only a sufficiently fine examiner reacts to the within-group changes the agent must track. As the examiner coarsens it grows blind to those changes and its $\rho^\chi_t$ falls below the true regret, an optimistic and invalid assessment that would wrongly certify the agent as near-optimal. This is the failure of the lower-bound and optimism assumptions when the examiner is too coarse for the world.

\textbf{An under-resolved agent reveals a structural gap.} When the examiner is fine-grained but the agent is restricted (Figure~\ref{fig:capacity}, bottom), the limitation is the agent's. Here $\rho^\chi_t$ stays a valid upper bound and settles just above the residual gap the restricted agent cannot close, a gap that grows as the agent coarsens, while a fully resolved agent drives both regrets toward zero. Perceived regret thus attributes the persistent regret to the agent's capacity rather than masking it.

\subsection{Comparing learners with a fixed examiner}
\label{sec:learner_comparison}

A natural use of perceived regret is model selection without ground truth: given one sufficiently powerful examiner, can we rank a pool of learners by $\rho^\chi_t$ and recover the ranking we would obtain from the unavailable true regret? We run several standard bandit learners in the switching world under a single matched examiner. We expect $\rho^\chi_t$ to upper-bound the true regret for each learner and, more importantly, to preserve their relative ordering.

\begin{figure}[t]
\centering
\includegraphics[width=\textwidth]{image/fig5_learner_comparison.pdf}
\caption{Perceived regret as a stream-only comparison metric across learners in the switching bandit. \textbf{(a)} For each learner, $\rho^\chi_t$ (blue) tracks and upper-bounds the oracle true regret (red); dotted lines mark segment boundaries. \textbf{(b)} Final perceived versus true regret over all learners and seeds; the strong rank correlation (Spearman $\rho_S = 0.85$) shows that ranking by $\rho^\chi_t$ recovers the true ordering without access to ground truth.}
\label{fig:learner_comparison}
\end{figure}

Both expectations hold. For every learner, $\rho^\chi_t$ tracks the true regret from above throughout learning (Figure~\ref{fig:learner_comparison}a). Aggregating each run to a single number, the final perceived and true regret are strongly rank-correlated across learners and seeds (Figure~\ref{fig:learner_comparison}b), so the examiner recovers the correct ordering without ever observing the truth. The scatter lies almost entirely above the diagonal, consistent with $\rho^\chi_t$ acting as a conservative proxy rather than an unbiased estimate.

\textbf{A recipe for unknown streams.} These experiments suggest a simple procedure when the world is unknown: instantiate a small, diverse pool of examiners whose biases bracket plausible dynamics and resolutions; report $\rho^\chi_t$ from each alongside the agent's behaviour; and treat agreement as evidence of meaningful estimates and disagreement as a sign that the examiner pool, not necessarily the agent, needs revisiting.
