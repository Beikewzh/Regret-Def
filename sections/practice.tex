\section{Perceived Regret in Practice}
\label{sec:practice}

In this section, we conduct simple experiments to illustrate how perceived regret can be used in practice and the effect of an examiner's modeling choices.

We consider a multi-armed Gaussian bandit where the optimal policy and the true regret are both known, so $\rho^\chi_t$ can be assessed against ground truth. 
We begin by conducting simple experiments to explore how the choice of examiner and its modeling biases impacts the perceived regret.
% We begin by showing how two instructive scenarios: an examiner reports a valid and tight $\rho^\chi_t$ when its assumptions match the environment's structure, and it may fail when they do not.
% Throughout, our expectation follows the theory of Section~\ref{sec:nfl}: an examiner reports a valid and tight $\rho^\chi_t$ when its assumptions match the environment's structure, and it may fail when they do not.
We consider two different environments: a stationary bandit and a nonstationary bandit, where the optimal action switches at regular intervals.
In each one, we run a \emph{good} examiner, whose modeling assumptions match the world's dynamics, and a \emph{bad} examiner, whose assumptions are mismatched. More precisely, the good examiner for the nonstationary environment is also aware of the switches but the bad examiner mistakenly assumes the environment is stationary. The good/bad examiners are defined in the reverse manner for the stationary environment.
Each examiner is run on two agents, one that is near-optimal in that environment and one that learns poorly.
Complete experimental details are in Appendix~\ref{app:details}. 


\textbf{A good examiner converges to the true regret.} 
% \wes{say something about what these agent and examiners are}
% the precise agents and examiners are listed in Appendix~\ref{app:details}.
When the examiner is good, its perceived regret is valid  in the sense of Proposition~\ref{prop:valid}: it stays above the true regret. Furthermore, it shrinks as the agent learns. 
In the stationary world (Figure~\ref{fig:practice_main}a), the good examiner's $\rho^\chi_t$ decays toward zero on the converging agent and settles at the residual sub-optimality gap on the agent that keeps exploring, tracking the true regret from above in both cases. 
In the switching world (Figure~\ref{fig:practice_main}b), the good examiner is reset-aware: its $\rho^\chi_t$ spikes at each change and recovers between changes, following the true regret of the agent that re-adapts, and it reports persistent regret for the agent that wrongly assumes the world is static.

\textbf{Failure modes of a bad examiner.} 
There are two primary ways an examiner can fail.\\
\textit{Invalid upper bound.} 
An examiner can produce a perceived regret lower than the true regret. 
We observe this in the switching world with the bad examiner on UCB1: 
The bad examiner assumes stationarity and keeps stale data, so its lower confidence value $\Vlow_t$ is held too high and its aspiration $\Vhi_t$ too low; its $\rho^\chi_t$ then drifts toward zero between changes and falls \emph{below} the true regret even while the agent is incurring large regret. 
This breaks the optimism assumption (Assumption~\ref{a:optimistic}), so the perceived regret is no longer valid and would wrongly certify the agent as near-optimal.\\
\textit{Non-decreasing upper bound.} The examiner may produce a perceived regret that is larger than the true regret but never decrease it. 
While not as dire as producing invalid regret bounds, this is also unsatisfying since it wrongly suggests that the agent can keep improving even though its policy may be optimal. 
This failure case is illustrated in the stationary world with the UCB1 agent. The examiner's perceived regret plateaus at a high value even though the agent achieves near-zero regret. This occurs since it mistakenly models the bandit as being nonstationary when it is not.

Taken together, the success and failure cases showcase the practical side of the no-free-lunch result of Section~\ref{sec:nfl}: an examiner is informative only when it carries the appropriate prior knowledge about the world and may be misleading when it does not. 
We further explore these considerations across another axis, varying the relative representational capacity of agent and examiner (results in Appendix~\ref{app:capacity}).

\subsection{Comparing learners with a fixed examiner}
\label{sec:learner_comparison}
In a complex world, we rarely have access to the true regret, yet we still want to compare candidate learners from a single stream of experience \citep{javed2025bigworld}. Perceived regret offers a stream-only surrogate: given an examiner, we can rank a pool of learners by their $\rho^\chi_t$ value. By using the same examiner, we can get a consistent measure of performance across different agents.

We run several standard bandit learners in the switching world under a single good examiner, and we expect $\rho^\chi_t$ to upper-bound each learner's true regret while giving meaningful performance estimates.

In Fig.~\ref{fig:learner_comparison}(a), we see that the perceived regret always upper bounds the true regret for different learning agents and decreases within each task.
Then, in Fig.~\ref{fig:learner_comparison}(b), we see that the perceived regret is highly-correlated with the true regret with a Spearman correlation of 0.87. 
For any fixed algorithm, we can see an even better agreement as the scatterplot shows the points lie roughly on a line.
These results suggest that perceived regret is a reasonable proxy for true regret in this known setting although we note that the main advantage of the perceived regret is that it can be computed in general environments where the true regret is unavailable.

\subsection{Evaluating Exploration with an Examiner}


We can consider another use-case of the examiner. Instead of evaluating the agent's value (its ability to collect reward), we can define a pseudo-reward and evaluate how well the agent accumulates that quantity. This can be viewed as a particular cumulant in general value functions \citep{sutton2011horde}.
As an example, we can quantify the agent's ability to explore by defining a quantity $r_{IG}$ representing the ``information gain" from choosing an action.

In the following bandit experiment, we use $r_{IG}(a) = C_k(a) - C_{k+1}(a)$ where $C_k$ is the width of the confidence interval after $k$ pulls of action $a$. 
This value quantifies how much tighter the confidence interval becomes after choosing that action. That is, how much the uncertainty has been reduced.

The examiner computes its perceived regret as before but we can now interpret its output $\rho^{IG}$ as an estimate of the difference between the maximum information gain the agent can obtain and how much  information it currently collects. 
Again, if the perceived regret (with respect to $r_{IG}$) is low, we can say that the agent is exploring well.
