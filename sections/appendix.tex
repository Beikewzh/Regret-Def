
\section{Theory}
\label{app:theory}
\nofreelunch*


\begin{proof}
    We proceed by contradiction and show that, given a history $h$ and an agent policy $\pi$, any examiner that produces a non-vacuous bound will be invalid in some environments. 
    Suppose we have an agent with history $h=\{o_0, ..., a_{t-2}, r_{t-1}, o_{t-1}\}$.
    and that the examiner $\chi$ produces a non-vacuous bound $\rho^\chi(h) < \rho_{bound}(h)$, where we make the perceived regret's dependence on the history explicit. 
    We will show that $V^{\pi} \le V_{unif}$ and $\sup_{\pi'} V^{\pi'} = V^{max}$. This will imply $V^*- V^\pi > \rho^\chi $  (perceived regret does not bound the true regret), establishing the contradiction.

    Given history $h$, we produce an environment $\Env_{bad}$ that generates the desired bounds. 
        
    The agent produces a distribution over its next action given its history. Let $a_{min}$ be the action with minimal probability (this exists since the action space is discrete).
    To define $\Env_{bad}$,
    we let $a_{min}$ have associated reward $\Rew(h, a_{min}) = R_{max}$ and 
    lead to a future where a unique optimal action at each step receives $R_{max}$. On each future step, this optimal action is the one that minimizes the policy's probability. 
    Thus, the optimal policy has value $V_{max}$ as it assigns probability 1 to $a_{min}$,the optimal action, at every step. 
    All other actions lead to 0 rewards forever.
    We also define $\Env_{bad}$ such that its reward  and observation functions are consistent with the history $h$. This is always possible since we put no restrictions on these functions and each element of the history $h$ depends only on the past. 

    Next, we show that $V^\pi(h) \le V_{unif}$ in $\Env_{bad}$. 
    The uniform random policy will have associated value $V_{unif}(h)= \frac{1}{|\Actions|} {R_{max}} + \gamma V_{unif}(h || a_{min}, R_{max},o' )$, where $a || b$ denotes the concatenation of history $a$ followed by $b$. This expressions is due to the fact that one action has reward $R_{max}$ and other actions leads to 0 reward forever.
    Then, consider the value of the policy $\pi$. Policy $\pi$ assigns probability $\pi(a_{min}) \le 1/|\mathcal{A}|$ to the optimal action.
    So, $V^\pi(h) = \pi(a_{min})R_{max} + V^\pi(h||(a_{min}, R_{max}, o') \le V_{unif} $ since $\pi$ assigns less (or equal) probability to each optimal action. 

    Thus, $V^\pi \le V_{unif}$ and $V^* = V_{max}$ as desired.
    
\end{proof}
\section{Experimental details}
\label{app:details}

\paragraph{World.} All bandit experiments use a bounded-Gaussian arm model: a pull of arm $i$ returns $r = \mathrm{clip}(\mathcal{N}(\mu_i,\sigma^2),0,1)$ with noise $\sigma = 0.1$. There are $N = 10$ arms whose means form a fixed ladder of values evenly spaced over $[0.1, 0.9]$, so adjacent arms differ by only $\approx \sigma$ and separating the best arm from its nearest competitors is a genuine statistical problem. The ladder is assigned to the arms by a random permutation, redrawn for every seed so that no two seeds face the same task. The stationary world keeps this assignment fixed for $T = 10{,}000$ steps; the switching world runs five segments of $10{,}000$ steps and re-permutes the ladder at each of the four boundaries, so the entire ranking of the arms changes between segments rather than only the identity of the best arm. All curves report the mean over $100$ seeds with $95\%$ bootstrap intervals, and per-step regret is shown after a moving-average smoothing.
\paragraph{Agents and examiners (Figure~\ref{fig:practice_main}).} Each examiner keeps, for every arm, an empirical mean $\hat\mu_i$ from $n_i$ observed pulls and a Hoeffding half-width $c_i = \sigma\sqrt{2\ln t / n_i}$, and reports the optimistic aspiration $\Vhi_t = \max_i(\hat\mu_i + c_i)$ together with the lower confidence value $\Vlow_t = \hat\mu_{a_t} - c_{a_t}$ on the played arm, so that $\rho^\chi_t = \Vhi_t - \Vlow_t$ (clipped to $[0,\Rmax]$ with $\Vhi_t \ge \Vlow_t$). The examiner variants differ only in which observations enter the counts: a full-memory bracket uses all past data, a sliding-window bracket uses the last $W$ steps, and a reset-aware bracket clears its counts at each known boundary. In the stationary world the well-adapted agent is UCB1 \citep{auer2002ucb} and the poorly-adapted agent is $\varepsilon$-greedy with $\varepsilon = 0.1$; the good (matched) examiner is the full-memory bracket and the bad (mismatched) examiner is the sliding-window variant that assumes the world keeps changing. In the switching world the roles invert: the well-adapted agent is a reset-UCB that resets at the known boundaries and the poorly-adapted agent is UCB1; the good examiner is reset-aware and the bad examiner is the full-memory bracket that assumes the world is static.

\paragraph{Capacity (Figures~\ref{fig:capacity} and \ref{fig:capacity_stationary}).} Capacity is varied by aggregating the arms into $G \in \{1,2,5,10\}$ equal groups, so the coarse party distinguishes only $G$ groups while the other stays per-arm. The agent is a sliding-window UCB and the examiner is reset-aware. In the forward setting the examiner is coarsened and the boundaries are placed so that a low-resolution examiner cannot see them; in the reverse setting the agent is coarsened and the examiner is per-arm.

\paragraph{Learner comparison (Figure~\ref{fig:learner_comparison}).} Five learners are run in a switching world whose arm set grows from $10$ to $30$ across segments, each assessed by the single reset-aware examiner: UCB1 \citep{auer2002ucb}, sliding-window UCB and discounted UCB \citep{garivier2011switching}, EXP3.S \citep{auer2002nonstochastic}, and Thompson sampling \citep{thompson1933likelihood}. The reported rank correlation is Spearman $\rho_S = 0.85$ over the $5\times100$ (learner, seed) pairs.

\paragraph{Intrinsic motivation (Figures~\ref{fig:intrinsic_beta} and \ref{fig:intrinsic_quality}).} The bonus added to the agent's reward is $\beta$ times the examiner's per-arm Hoeffding confidence width $c_i(t)=\sigma\sqrt{2\ln t_{\mathrm{eff}}/(n_i+1)}$, clipped into $[0,1]$, with no other novelty term. The world is a growing-action switching bandit (arm set $10 \to 50$). The good examiner is reset-aware; the ensemble averages three weak count trackers (a sliding window, a full-memory tracker, and a sliding window with a mis-specified $\sigma$); the vacuous examiner returns a uniform, count-independent width $\sigma\sqrt{2\ln t}$ for every arm. We sweep $\beta \in \{0.5, 1, 2\}$, with $\beta=0$ the no-bonus reference.





\section{Exploration via examiner uncertainty}
\label{sec:intrinsic_optimization}


The examiner has so far been a passive scorer, but its uncertainty can also drive the agent. We add to the agent's reward an intrinsic bonus equal to the examiner's per-arm confidence width, scaled by $\beta$, with no other novelty signal, and study a switching world whose action set grows over time. Treating an uncertainty estimate as an exploration bonus is the basis of count-based and optimism-driven exploration and of intrinsic motivation more broadly \citep{strehl2009pacmdp, auer2008near, oudeyer2007intrinsic, lidayan2025bampf}; here the estimate is supplied by the examiner rather than the agent, which lets us ask whether a good examiner can double as a useful uncertainty metric.


It can, when the examiner is good. A reset-aware examiner's uncertainty rises again at each change and prompts the agent to re-explore exactly when the world has shifted, which lowers cumulative regret well below the no-bonus baseline and speeds recovery after each switch (Figure~\ref{fig:intrinsic_quality}). The benefit is not unconditional, however: because the bonus persists, an examiner that keeps reporting uncertainty after the agent has converged will keep pushing it to explore and so erode performance, the familiar tension that an intrinsic bonus must eventually decay or be annealed \citep{lidayan2025bampf}. In our setting this appears as an over-exploration regime once $\beta$ is too large, which we report in Appendix~\ref{app:intrinsic_ablation}.

In practice we do not know the true dynamics, so we cannot guarantee a perfectly matched examiner, but we usually have some partial knowledge. Averaging a small, diverse pool of reasonable but imperfect examiners turns that partial knowledge into a robust bonus: an ensemble of three weak examiners reduces regret almost as much as the single oracle examiner, without knowing when or how the world changes (Figure~\ref{fig:intrinsic_quality}). What must be avoided is an examiner with no real knowledge of the world: a vacuous examiner whose uncertainty is uniform across arms carries no exploration signal, and as $\beta$ grows its bonus saturates the reward until learning collapses to the random-policy level (Figure~\ref{fig:intrinsic_quality}).
A single weak examiner is often sufficient on its own, but it is not guaranteed to be, 
and \wes{fix ref} shows one whose mis-calibration degrades sharply; the ensemble is therefore best understood as a hedge against a poorly chosen examiner rather than as a free improvement over a good one.

These observations suggest a simple procedure for unknown streams: instantiate a small, diverse pool of examiners whose biases bracket plausible dynamics and resolutions, report $\rho^\chi_t$ from each alongside the agent's behaviour, and treat agreement across the pool as evidence of meaningful estimates and disagreement as a sign that the pool, not necessarily the agent, needs revisiting.

Two ablations support the discussion. First, sweeping the bonus strength with the good examiner (Figure~\ref{fig:intrinsic_beta}) shows that a moderate $\beta$ lowers regret while too large a $\beta$ over-explores, because the bonus keeps pushing an already-converged agent to explore. Second, at the fixed strength $\beta=1$ used in Figure~\ref{fig:intrinsic_quality}, we compare the three weak examiners individually against their ensemble (Figure~\ref{fig:single_examiners}). The weak examiners disagree: the sliding-window and full-memory trackers each lower regret below the no-bonus baseline, but the mis-calibrated one, with an oversized $\sigma$, raises it above. Since we cannot know in advance which weak examiner suits an unknown world, averaging them is the safe choice: the ensemble stays below the baseline and close to its better members, absorbing the bad one rather than inheriting its failure.

\section{Relative capacity between learner and examiner}
\label{app:capacity}

A further axis, orthogonal to the dynamics mismatch of Section~\ref{sec:practice}, is representational capacity: how finely the examiner, or the agent, resolves the action space. We probe it by letting one party distinguish only a coarse grouping of the arms while the other stays fine-grained. We expect $\rho^\chi_t$ to stay valid and tight only when the examiner is at least as fine-grained as the distinctions that drive regret, and, when the agent is instead the coarser party, to remain valid while exposing the agent's structural limitation.

\begin{figure}[t]
\centering
\includegraphics[width=\textwidth]{image/fig4_capacity_forward.pdf}\\[6pt]
\includegraphics[width=\textwidth]{image/fig4_capacity_reverse.pdf}
\caption{Relative capacity in the switching bandit, one cell per resolution, labelled by group size (actions per group). \textbf{(Top, forward).} A fine-grained agent assessed by an examiner that resolves only $G$ groups: a coarse examiner is blind to within-group changes and under-reports, with $\rho^\chi_t$ falling below the true regret (red), while only the fully resolved examiner tracks it. \textbf{(Bottom, reverse).} An agent restricted to $G$ groups assessed by a fully resolved examiner: $\rho^\chi_t$ plateaus just above the structural gap (labelled) the restricted agent cannot close, while the fully resolved agent drives both regrets toward zero. Red: true regret; blue: $\rho^\chi_t$.}
\label{fig:capacity}
\end{figure}

\textbf{An under-resolved examiner under-reports.} When the agent is fine-grained but the examiner aggregates arms (Figure~\ref{fig:capacity}, top), only a sufficiently fine examiner reacts to the within-group changes the agent must track. As the examiner coarsens it grows blind to those changes and its $\rho^\chi_t$ falls below the true regret, an optimistic and invalid assessment that would wrongly certify the agent as near-optimal. This is the failure of the lower-bound and optimism assumptions when the examiner is too coarse for the world.

\textbf{An under-resolved agent reveals a structural gap.} When the examiner is fine-grained but the agent is restricted (Figure~\ref{fig:capacity}, bottom), the limitation is the agent's. Here $\rho^\chi_t$ stays a valid upper bound and settles just above the residual gap the restricted agent cannot close, a gap that grows as the agent coarsens, while a fully resolved agent drives both regrets toward zero. Perceived regret thus attributes the persistent regret to the agent's capacity rather than masking it.

We repeat the capacity probe in the stationary world, where the effect is milder because there are no changes to expose a coarse examiner. Figure~\ref{fig:capacity_stationary} shows both directions. In the forward case, once the agent has identified the optimal arm the examiner's aggregation barely matters: all resolutions $G$ converge to nearly the same small $\rho^\chi_t$, since the only distinction that drives regret (optimal versus sub-optimal arm) is preserved under every grouping, with the early-transient separation visible in the inset. In the reverse case, a capacity-restricted agent again plateaus at the structural gap induced by its grouping, and the fully resolved examiner's $\rho^\chi_t$ tracks that gap from above. The stationary world therefore exercises the same mechanism as Figure~\ref{fig:capacity} but without the non-stationary stress that makes an under-resolved examiner visibly invalid.
